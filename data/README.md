
# Data Folder

This folder contains the sample metadata used for the differential gene expression analysis.

## Files Included

- Sample_Information.csv

## Note

Large intermediate files (Expression_Matrix.csv, Lung_Data.RData, and GSE19804_GEO_Object.RData) are not included in this repository because they exceed GitHub's web upload size limit (25 MB).

These files can be regenerated by running the analysis script (`dge_analysis.R`) using the publicly available GEO dataset GSE19804.
